Breaking Down The Header

The header can be seen to contain three components which are;

  • The Parent: Embedding two child Components [Child 1 and Child 2].
  • Child 1: Holding the brand logo/name.
  • Child 2: Holding a list of navigations.

Each of this components are enclosed in <div>[short for division] tags. In the above case, we have a parent div nesting two child divs.

The primary purpose of a <div> is to act as a container for grouping other HTML elements together. <div> is a block-level element, which means it takes up the full width available to it and starts on a new line, giving it it's popularity amongs programmers.
