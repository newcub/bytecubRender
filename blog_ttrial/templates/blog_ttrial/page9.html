Responsive Design

Introduction

Responsiveness in web design referes to the ability of a website to adapt it's layout and content to different screen sizes and orientations. This ensures a seamless user experience accross various devices, such as desktops, tablets, and smartphones.

If we try resizing our computer screen, we would notice that the header of our blog website breaks and become unappealing when the screen size becomes very small. In this chapter, we are going to fix that.

@media Queries

The @media query in CSS is a powerful tool for creating responsive designs. It allows you to apply different styles based on the characteristics of the device, such as it's width, height, orientation and resolution.

How @media Queries Are Used

For example @media(max-width: 830px) Allows the developer to apply specific CSS styles when the viewport width is 830px or less.

@media(min-width: 830px) Allows the developer to apply specific CSS styles when the viewport width is 830px or more.
