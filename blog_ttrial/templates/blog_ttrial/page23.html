A Responsive Design

Redefining The Grid

Shrinking our webpage to fit smaller screens, it breaks and become very ugly. To fix this, we would have to redefine the webpage's layout on smaller screens.

style.css [@media]

@media(max-width: 840px){
.main-container{
grid-template-columns : 1fr;
}
}

Result [On Smaller Screens]

grid-template-columns : 1fr; collapses the innitial two columns layout into a single column.

Notice that the blog post comes before the informative elements[About Us, Categories and Follow Us] that were once on the Sidebar. This is somehow inapropriate and unproffesional.

To revert their possitions, we are going to use the order property in CSS.

style.css [@media]

@media(max-width: 840px){
.main-container{
grid-template-columns : 1fr;
}
.side-bar{
order : -1;
}
}

Result [On Smaller Screens]

The order property in CSS is used to control the order in which items appear within a container.

This property is particularly useful when you want to rearrange items visually without changing the HTML structure.

Note:

The order property only work for items within a grid or flex box container.
