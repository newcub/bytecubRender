overflow-x: property specifically controls the horizonal overflow [a situation where the width of a child container exceeds the parent's].

The auto value tells the browser to provide a horizontal scrollbar only if the children overflows the parent's width. If the children fits within the parent's width, no scrollbar will be displayed.

flex-basis: 80%; ensures that a child element innitially occupy 80% of the containers available space.

flex-grow: 0; ensures that a child element never grow to take up extra space, even if there's room.

flex-shrink: 0; ensures that a child element never shrink, even if there isn't enough room in the container.

min-width: 80%; though not really neccesory because of the flex-basis property above, acts as an extra layer to ensure that the container width doesn't go below 80% incase the browser miscalculates flex-basis: 80%; in certain situations.
