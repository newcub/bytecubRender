Final Touches On The Blog Post

The Body

Let's begin by adding a background color to the body to create a contrast between the rest of the webpage and the blog post.

Just below the *{css code} selector, add the following lines of code.

CSS

*{
/* CSS code */
}
body{
background-color : #f9f9f9;
color : #333;
line-height : 1.6;
}
