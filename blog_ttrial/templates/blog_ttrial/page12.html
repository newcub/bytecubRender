CSS Grid

Introduction

CSS Grid layout is a two dimensional layout system for the web. It allows developers to create responsive websites by giving them full control over the rows and columns.

Unlike Flexbox which is one-dimensional, Grid can handle both rows and columns simultaneously.

How It's Used

The Parent element becomes the grid container, and it's children are grid items that can be manupulated.

display: grid; is used to create a grid layout for the parent element [main-container].

grid-template-columns: 3fr 1fr; specifies that the grid should have two columns with a fractional unit [fr] distribution.

3fr 1fr ensures that, the first column [blog-posts] occupies 3/4 of the Parent container while the second column [side-bar] occupies 1/4 of the Parent container.
