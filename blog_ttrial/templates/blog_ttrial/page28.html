In the grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)); expression,

  • The repeat() function allows you to define a pattern for the columns.
  • The auto-fit tells the browser to automatically fit as many colums as possible into the available space.
  • The minmax() function defines a size range for the colums.
  • The first value, 300px, is the minimum width of the column. This means that each column will not be smaller than 300 pixels.
  • The second value, 1fr, is the maximum width of each column. The fr stands for fraction of the available space.

To better understand this concept, try resizing your viewport/screen-size and see how it works.

Note:

This is the best way of building responsive grid layouts, as it saves us the hasle of using the @media query to define layouts for different screen sizes.

Though at screen sizes less than 300px the structure might break. Details about this and how to fix it would be discussed in the next chapter [Beauty page].
