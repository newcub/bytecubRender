Key Points

Many HTML elements, such as headings, paragraphs, and lists, come with default padding and margin. By adding padding: 0; and margin: 0; you ensure that elements do not have unwanted spacing, allowing for a more controlled layout.

Similarly, some elements may have default borders [like buttons or input fields]. Setting border: none; removes default borders, giving you freedom to style them as you wish.

The box-sizing: border-box; property changes the box model calculation. With this setting, the width and height of elements include padding and borders, making it easier to size elements without having to account for additional padding or borders.
