Font Awsome

Font Awsome is an internet icon libary and toolkit, used by millions of designers, developers and content creators.

The social media icons on the Follow Us widget are going to be done using fontawsome's icon libary.

How Do I Include Font Awsome On A Webpage?

There are so many ways in adding fontawsome to a webpage, but in this tutorial, we are going to be using the method of linking our webpage to the font-awesome.min.css file over the internet.

By including <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css"> in the head of our webpage, we should be able to access the icons in the Font Awsome icon libary.

How Do I Insert An Icon?

To insert an icon from the fontawsome libary, add the name of the icon class to any inline HTML element. The <i> and <span> elements are widely used to add icons.

For example, <i class="fa fa-facebook-square"></i> displays

Visit their official website [fontawsome.com] to gain access to their entire icon libary by their icon class names.

Note:

Icon size and color can be changed using CSS styling.
