Child One

Looking closely at the image above we'd see that, within the child One container [blog-posts class], we have a sub-container [marked in red] which houses the blog elements; image, an h1 heading [title], two paragraphs [one containing the publishing info and the other containing the description of the post], and a link saying Read More.

This sub-container [let's call it blog-post] is a stand-alone component and can be reused independently. That is, it would be multiplied to generate multiple blog-posts [reason we named it blog-post without the 's'].

Instead of just using another <div> to create this sub-container let's use the <article> tag which is popularly used by programmers to create independent reusable content.

margin: 2px; was added to separate the individual blog-post from eachother and make them clearly visible.

Note:

A <div> could still be used in place of an <article> but it is a universal HTML block tag [as it has no specific semantic meaning] and wouldn't standout at first glance as a reusable code-block to other programmers reviewing your code or even screen readers.
