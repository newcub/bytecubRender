For Loops

A for loop is a control structure in JavaScript that allows you to execute a block of code a specific number of times. It is particularly useful when you know in advance how many times you want to iterate through a block of code. The for loop consists of three main parts: initialization, condition, and increment/decrement. Each of these parts are seperated from eachother by a semi-colon [';'].

  • initialization: This is where you declare and initialize a loop control variable [e.g., let i = 0].
  • condition: This is a boolean [true or false] expression that is evaluated before each iteration. The loop continues as long as this condition is true.
  • increment/decrement: This is where you update the loop control variable [e.g., i++ to increment or i-- to decrement].
