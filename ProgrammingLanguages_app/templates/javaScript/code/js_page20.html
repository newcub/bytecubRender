Exercise

Now that you have learned about functions in JavaScript, here are two exercises for you to try:

1. Create a Function to Calculate Area:

  • Write a function called calculateArea that takes two parameters: length and width.
  • The function should return the area of a rectangle [length * width].
  • Use prompt() to get the length and width from the user, call the function, and log the result.

2. Create a Function for Temperature Conversion:

  • Write a function called convertToFahrenheit that takes one parameter: celsius.
  • The function should convert Celsius to Fahrenheit using the formula: F = C * 9/5 + 32 .
  • Use prompt() to get the Celsius temperature from the user, call the function, and log the result.
