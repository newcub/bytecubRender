Nested if Statements

Nested if statements are if statements placed inside another if statement. This allows you to check multiple conditions in a hierarchical manner. You can use nested if statements when you need to evaluate a condition only if a previous condition is true.

Output

>You can drive.   if the first and second conditions are true.

>You need a driver's license to drive.   if the first condition is true but the second is not.

>You are not old enough to drive.   if the first condition is false.

In the above example:

  • The first if checks if the user is 18 or older.
  • If true, it then checks if the user has a driver's license.
  • Depending on the answers, it provides appropriate messages.

When to Use Nested if Statements

Nested if statements are useful when:

  • You have multiple layers of conditions to check.
  • The second condition should only be evaluated if the first condition is true.

However, be cautious with nesting too deeply, as it can make your code harder to read and maintain. In such cases, consider using logical operators or restructuring your code for clarity.
