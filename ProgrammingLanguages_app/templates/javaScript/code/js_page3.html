Data Types: Number

What Are Data Types?

In javaScript, data types are categories of values. The main types include:

  • String: Text [e.g "Hello"]. Strings are usually declared in double qoutes [" ... "].
  • Number: Numeric values [e.g 42]. Numbers don't need any qoutes around them.
  • Boolean: true or false.

The Number Data Type

In javaScript, the number data type is used to represent whole numbers [intergers] and decimal [floating point values]. This makes it a versatile type for performing calculations and handling various mathematical operations.

Arithmetic Operations on Numbers

JavaScript supports several arithmetic operations that allow you to perform calculations easily:

  • Addition (`+`): Combines two numbers.
  • Subtraction (`-`): Finds the difference between two numbers.
  • Multiplication (`*`): Calculates the product of two numbers.
  • Division (`/`): Divides one number by another.
  • Modulus (`%`): Returns the remainder of a division operation [e.g. 5 % 2 returns 1 ]. Take note of the modulus operator because we are going to be using it alot in the future.
