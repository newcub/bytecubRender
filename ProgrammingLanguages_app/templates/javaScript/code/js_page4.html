Data Types: Strings

What is a String?

A string in javaScript is a series of characters used to represent text. Strings can include letters, numbers, symbols, and spaces.

In javaScript ,strings are always enclosed in qoutes.

Using Qoutes with Strings

You can create a string using either double qoutes [" ... "] or single qoutes [' ... ']. Both types of qoutes work thesame way, so you can use whichever you prefer.

However, it's important to be consistent throughout your code.

Example

{% for snippet in code_1 %} 
                        
                        
                        {{ snippet.get_highlighted_code.html | safe }}
{% endfor %}

Output

> Hello, World!

> Goodbye, everyone!

Why Use Qoutes?

Qoutes are necessary because they tell javaScript where the string starts and ends. If you forget to use qoutes or use them incorrectly, you will get an error.

If you want to include a qoute inside a string, you can use the opposite type of qoute to avoid confusion:

{% for snippet in code_2 %} 
                        
                        
                        {{ snippet.get_highlighted_code.html | safe }}
{% endfor %}

Output

> She said, 'Hello!'

> He replied, "Hi!"

You can also, declare and empty string and later on assign a value to it, e.g let name = "";

Concatenating Strings

Concatenation is the process of joining two or more strings together to create a new string. You can concatenate strings using the + operator.
