Understanding Variables in JavaScript

Though not a must, it is good practice to end each statement with a semicolon [;] in javaScript.

What Are Variables?

In JavaScript, a variable is a way to name and store a value so you can use it later in your code. For example, if you want to remember a person's name or their age, you can use variables to store that information.

How to Create Variables

You can create a variable using one of three keywords: var, let, or const. Here’s a simple breakdown:

1. var: This is the oldest way to create a variable. It can be used anywhere in your code, but it can sometimes lead to confusion because of its scope [where it can be accessed].

Comments

Just to make a quick note, comments in javaScript explain your code, making it easier for someone going through it to Understand.

Use // for single line comments and /* ... */ for multi-line comments.

They are ignored by the browser, so they don't affect how your code runs.
