What is an Object?

In JavaScript, an object is a standalone entity, with properties and type. It is similar to real-life objects, like a car or a dog, which have characteristics [properties] and behaviors [methods].

Objects are very useful when designing real life models, like a character's behavior/attributes in a video game.
