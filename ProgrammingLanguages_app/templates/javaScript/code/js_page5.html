What is a Boolean?

In javaScript, a Boolean is a data type that can hold one or two values: true or false. These values are used to represent truth values, which means they can indicate whether something is correct or incorrect, yes or no, on or off, etc.

When to Use Booleans?

Booleans are commonly used in conditional statements. That is, in situations where we might want to perform a specific task when a certain condition is met or not.

For example, we might want to login a user into an account if they provide the right email and password and if not, we might display 'wrong password, try again'.
