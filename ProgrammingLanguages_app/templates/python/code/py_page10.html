If Statements in Python

What is an If Statement?

An if statement is a fundamental concept in programming that allows you to execute a block of code based on a certain condition. It helps you make decisions in your code. If the condition evaluates to True, the code inside the if statement will run; if it evaluates to False, the code will be skipped.

Indentation

In Python, indentation is crucial. It defines the block of code that belongs to the if statement. All lines of code that should run if the condition is true must be indented with spaces or tabs (but not both).

Comparison Operators

When working with conditions, you often use comparison operators. Here are some common ones:

  • ==: Equal to
  • !=: Not equal to
  • >: Greater than
  • <: Less than
  • >=: Greater than or equal to
  • <=: Less than or equal to

Logical Operators

You can also combine conditions using logical operators:

  • and: True if both conditions are true.
  • or: True if at least one of the conditions is true.
  • not: Inverts the truth value of a condition.
