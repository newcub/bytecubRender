Tuples

Introduction

A tuple is a built-in data structure in Python that is similar to a list but has some key differences. Tuples are used to store multiple items in a single variable. The main characteristics of tuples are:

  • Immutable: Once a tuple is created, you cannot change its elements (that is, you cannot add, remove, or modify items).
  • Ordered:The items in a tuple have a defined order, and you can access them using indexing.
  • Heterogeneous:Tuples can contain elements of different data types (e.g., integers, strings, lists, etc.).

In summary, tuples are versatile data structures that allow you to store multiple items in an ordered and immutable way. While they differ from lists in terms of mutability (tuples cannot be modified after creation), many operations that can be done on lists can also be performed on tuples, such as looping, indexing and slicing.
