HTML provides various ways to create lists, the most common being:

5.1 Unorderd Lists (ul):

Used to create a bulleted list of items. Each item is denoted by the <li>[list item] tag wraped in the <ul> tag.

Code example

{% for snippet in code_1 %} 
                        
                        
                        {{ snippet.get_highlighted_code.html | safe }}
{% endfor %}

Result

  • First Item
  • Second Item
  • Third Item

5.2 Orderd Lists (ol):

Used to create a numbered list of items. Each item is denoted by the <li>[list item] tag wraped in the <ol> tag.

Code example

{% for snippet in code_2 %} 
                            
                            
                            {{ snippet.get_highlighted_code.html | safe }}
{% endfor %}

Result

  1. First Item
  2. Second Item
  3. Third Item

5.3 Definition Lists (dl):

Used to create a list of terms and thier definitions. The <dt>[definition term] tag is used for the definition while the <dd> term is used to contain the term being defined.

Code example

{% for snippet in code_3 %} 
                                
                                
                                {{ snippet.get_highlighted_code.html | safe }}
{% endfor %}

5.4 Nested Lists:

Putting a second list inside an <li> tag creates a sub-list in it called a nested list.
