HTML Tables

6.1 Introduction to HTML Tables

HTML tables are created to represent data in a structured format, organising information into rows and columns, much like a spread sheet. This allows for easy represntation and easy comparison of data.

6.2 Basic Structure of an HTML Table

  • <tr>[Table Row]: Defines a row in the table.
  • <th>[Table Header]: Defines a header cell in the table.
  • <td>[Table Data]: Defines a standard cell in the table that contains data.

The border attribute tells the web browser to draw lines around the entire table and between each cell [ The boxes inside the table ].

The number [or value] choose for the border can be changed. A larger number means thicker lines [e.g., border = "2" for a slightly thicker line].

6.2 Column Span

We may need entries in a table to stretch more than one column cell sometimes.

Thecolspan attribute is used in the <td> or <th> elements to specify how many colums a table should span.
