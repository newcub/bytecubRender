CREATING A WEBPAGE ON A PC

1.1 Using NotePad/NotePad++

Go to start, type Notepad or Notepad++ then click open

Copy the code on the right in Notepad or Notepad++

After copying the code, press Ctrl + S and choose a location to store the file.

After choosing a location, name the file as it suits you. In the case of this tutorial, it was named example.html

After having named the file, click save to Save it.

Note:

Adding .html to every html file name is mandatory as it tells the computer that it is an html file.

Go to the directory where you saved the file[example.html], right click on it and open with a browser of your choice.

Congratulations, you just created your first webpage.

1.2 Using Sublime Text

Why use Sublime Text?

  1. Sublime text provides a better graphics quality than Notepad and Notepad++ making it more user-friendly.
  2. Provides syntax highlighting.
  3. Has code correction features making coding faster.
  4. Moreover, it is lighter than other code editors out there, easy to use and free, making it just the best tool for begginers.

Click the download button below to download Sublime Text or download from thier official website.

After installing Sublime Text on your PC, open it as seen on the right.

Press Ctrl + N and copy the code on your right into your code editor.

After copying the code, press Ctrl + S and choose a location to save the file.

After choosing a location, name the file as it suits you. In the case of this tutorial, it was named example.html

After having named the file, click save to Save the file.

Note:

Adding .html to every html file name is mandatory as it tells the computer that it is an html file.

Go to the directory where you saved the file[example.html], right click on it and open with a browser of your choice.

Congratulations, you just created your first webpage.
