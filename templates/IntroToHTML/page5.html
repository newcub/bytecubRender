4.1 Introduction to HTML Paragraphs

HTML Paragraphs are fundamental building blocks for structuring text content on web pages. They are defined using the <p> tag. This tag helps to seperate blocks of text, making it more readable and organised for users. An example of how a paragraph is declared can be seen below.
