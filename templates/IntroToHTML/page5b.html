4.2.1 <center>:

The <center> Content </center> tag is used to center-align text and elements within a webpage.

4.2.2 Comment [<!-- First section -->]:

Comments in HTML are used notes or explanations within the code that are not displayed in the browser. Their purpose is to help developers understand and maintain their code by providing context and details.

4.2.3 <section>:

The <section> Content </section> tag is used to group content into different independent sections.

4.2.4 <i>:

The <i>italic</i> tag is used to emphasize text, typically rendering it in italics. This can be useful for highlighting specific words or phrases within a paragraph.

4.2.5 <br>:

The line break tag is used to insert a line break [line spacing] within elements in a webpage. In the above case, it creates a spacing between the two sections.

4.2.6 <strong>:

This tag is used to indicate that the text is of strong importance. By default, it renders the text in bold, which can help emphasize keypoints within a paragraph.
