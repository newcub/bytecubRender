7.1 Links in HTML

Links are created using the <a>[anchor] element. Users can click on anything between the opening <a> tag and the closing </a> tag. The primary attribute for linking is href, which specifies the URL of the page or the resource the link points to.

7.2 Filepaths in HTML

Filepaths are used to specify the location of files on the server or in the web project. There are two types of filepaths: absolute and relative.

1- Absolute Filepaths:

Absolute filepaths provide the full URL to a resource. Example, we want to access an image [photo.jpg] in the images folder of a website called 'example.com':

<img src="https://www.example.com/images/photo.jpg">

2- Relative Filepaths:

It is a path relative to the current document's location. Example, we want to access an image [photo.jpg] in the the same folder that we have the HTML document we are currentlly working on:

  • Project Folder
    • index.html
    • images
      • Photo.jpg

The corresponding HTML [in 'index.html'] is as follows:

<img src="images/photo.jpg">
