What is HTML?

HTML is like the skeleton of a webpage. It gives structure to your content, allowing browsers to understand how to display text, images, links, and more.

The Structure of HTML

Every HTML document is build upon a simple structure. Here's what the structure of an HTML page looks like:

<!DOCTYPE html>
<html>
<head>
  <title>My HTML Page</title>
</head>
<body>

</body>
</html>

Breaking it Down:

<html>, <head>,<body>, and so on... are called Tags.

Tags carry specific information within them, they are declared by <tag> [openning tag] and</tag>[closing tag].

They act like containers and tell the browser about the kind of information that lies within them.

For example, <p> This is a paragraph </p> tells the browser that what is within the <p> tag is a paragraph.

<!DOCTYPE html>: This declaration tells the browser to expect an HTML5 document.

<html>: The root element that wraps all your content.

<head>: Contains meta-information about your document [like title, character set, and linked styles].

<title>: The title appears in the browser tab, providing a clear indication of the page's content.

<body>: This is where you place all your visible content.
