7.2 Filepaths in HTML

Filepaths are used to specify the location of files on the server or in the web project. There are two types of filepaths: absolute and relative.

1- Absolute Filepaths:

Absolute filepaths provide the full URL to a resource. Example, we want to access an image [photo.jpg] in the images folder of a website with domain name 'example.com':
<img src="https://www.example.com/images/photo.jpg">.

2- Relative Filepaths:

Relative a path relative to the current document's location. Example, we want to access an image [photo.jpg] in the the same folder that we have the HTML document we are current working on:
<img src="images/photo.jpg">.
