8.1 Basic Image Styling Techniques

1. Sizing Images:

Use the height and width properties to control the dimensions of an image.

Example

img{
width: 100%;
height: auto;
}

Result

width: 100%; Ensures that the image fills the entire width of the container [using % makes the image responsive].

height: auto; maintains the image's aspect ratio no matter how much the width is stretched.

2. Borders:

Add borders to images using the border property. You can customize the border's width, style, and color.

Example

img{
border: 2px solid green;
border-radius: 8px;
}

Result

The border-radius property creates rounded corners around the image.

3. Box Shadow:

Create depth and dimension with the box-shadow property. This adds a shadow effect around the image.

Example

img{
box-shadow: 0 4px 8px gray;
}

Result

4. Opacity:

Adjust the transparency of an image using the opacity property.

Values range from 0 [completely transparent] to 1 [fully opaque].

img{
opacity: 0.7;
}

Result

5. Filters:

Apply visual effects to images using the filter property. Common filters include blur, brightness, and grayscale.

img{
filter: grayscale(100%);
}

Result
