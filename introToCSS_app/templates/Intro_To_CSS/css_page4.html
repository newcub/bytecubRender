CSS Box Model

Featured Video

Video Description:

The CSS Box Model is a fundamental concept in the design and layout.

It describes how elements on a webpage are structured and spaced. Understanding the box model is essential for effective CSS styling and layout management.

4.1 Components of the Box Model

1. Content:

This is the actual content of the box, such as text, images, or other media.

The dimensions of the content area are defined by the height and width properties.

2. Padding:

Padding is the space between the border and the box.

It creates an inner space within the box, enhancing the visual separation of the content from the box's edges.

Padding can be set using properties like padding-top, padding-bottom, padding-left, and padding-right, or the shorthand padding.

3. Border:

The border is a line that sorrounds the padding[if any] and content.

It can be styled using properties such as border-width, border-radius, border-style, and border-color.

4. Margin:

Margin is the outer space that surrounds the border of the box and separates it from other elements.

It creates space between the boxes of different elements.

Margin can be set using properties like margin-top, margin-bottom, margin-left, and margin-right, or the shorthand margin.

Example

Lets look modify the article from Exercise One.

CSS

article{
padding: 20px;
margin-bottom: 20px;
border-radius: 5px;
}

4.2 Box-Sizing Property

The box-sizing property changes how the width height of an element are calculated:

  • content-box: The default value. Width and height apply to the content box only.
  • border-box: Width and height include padding and border, making layout easier.

Example

*{
box-sizing: border-box;
}
