Adding CSS To a Webpage

Adding CSS to a webpage can be accomplished through several methods, each with its own use cases and benefits. Here are some primary ways to incorperate CSS into your HTML document

1.1 EXternal CSS

It involves linking to a separate file that containes all the styles for the webpage. This is the most common method, especially for larger websites, as it allows for easier maintenance and reusability accross multiple pages.

To style a webpage using external CSS, first create a separate file and name it with the .css extension [for example style.css]

style.css

p{
color: orange;
}

Key Points:

The <link> element can be used in and HTML document to link different files that we want to work with together. In this case, we want to see how we can link a CSS file using this element.

It should be noted that, the <link> element does not have a closing tag, and it is placed inside the <head> of an HTML document. It should use three attributes:

  1. href:This specifies the path to the CSS file.
  2. type:It specifies the type of document being linked to. The value should be text/css as seen above.
  3. rel:It stands for relationship and it specifies the relationship between the HTML document and the file being linked to. The value should be stylesheet when linking to a CSS file.

1.2 Internal CSS

Internal CSS is defined within a <style> tag in the <head> section of the HTML document.

It cannot be reused accross multiple HTML documents since it is built inside the document itself.

It involves adding the CSS styles directly within an HTML element using the style attribute.

This method is useful for applying unique styles to a single element without affecting others.
