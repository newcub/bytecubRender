CSS Selectors

CSS selectors are patterns used to target and style HTML elements in a webpage. Mastering CSS selectors is key to applying styles effectively accross a webpage.

2.1 Types of CSS Selectors

  • Universal Selector[*]: Selects all element in the document.

Example

*{
font-family: sans-serif;
}
  • Element Selector: Selects all elements of a specific HTML type.

Example

p{
font-size: 20px;
}

In the above example, all <p> elements within the HTML document are targeted.

  • Class Selector[.]: Selects all elements of a specific class attribute.

Example

.container{
background-color: blue;
}

In the above example, all elements with the class attribute container, are targeted.

  • ID Selector[#]: Selects all elements of a specific ID attribute.

Example

#amount{
font-size: 50px;
}

In the above example, all elements with the ID attribute amount, are targeted.

  • Descendant Selector: Selects all elements that are descendants of a specified element.

Example

.container p{
font-size: 50px;
}

In the above example, all <p> elements within container class , are targeted.

  • Child Selector: Selects all elements that are descendants of a specified element.

Example

li > a{
text-decoration: none;
}

In the above example, any <a> elements that are within the <li> element are targeted but not other <a> elements in the page.

  • Adjacent Sibling Selector: Selects an element that is immediately next to a specified element.

Example

h2 + p{
margin-top: 0;
}

In the above example, the <p> element that is directly next to the <h2> element is targeted but not other <p> elements in the page.

  • General Sibling Selector: Selects all siblings of a specified element.

Example

h2 ~ p{
color: red;
}

In the above example, all the <p> elements that are siblings of the<h2> element are targeted.
