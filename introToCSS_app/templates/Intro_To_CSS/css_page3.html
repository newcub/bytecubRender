Text Styling

{%if has_answered == True%} {%else%} {%endif%}

CSS provides various properties to style text in webpages. The properties allow you to control the appearance of text including color,spacing, font, size, and alignment.

In this chapter, we are going to be looking at the key text styling properties in CSS.

3.1 Font Properties

In this section, we are going to be working with the following HTML: <p>This is a Paragraph</p>

  • font-family: Specifies the font of the text.

Example

p{
font-family: Times;
}

  • font-size: Sets the size of the text.

Example

p{
font-size: 30px;
}

Result

  • font-weight: Controls the thickness of the text. Values can be normal, bold, bolder, or a number from 100 to 900.

Example

p{
font-weight: bold;
}

Result

  • font-style: Defines the style of the text as normal, italic, or oblique.

Example

p{
font-style: italic;
}

3.2 Text Color and Background

Colors in CSS can be set using a variety of formats, which include Color name, Hexadecimal, RGB, etc.

  1. Color Name: You can use predefined names in CSS. For example: color:orange;
  2. Hexadecimal: Hex codes are a popular way to define colors. For Orange, you would simply use: color:#FFA500;
  3. RGB[Red, Green, Blue]: The RGB format allows you to define colors using their red, green, and blue components. For oronage, it would be: color: rgb(255, 165, 0);

Lets use the class attribute in this section. Here is the HTML: <p class="text">This is a Paragraph</p>

  • color: Sets the color of the text.

Example

.text{
color: orange;
}

  • background-color: Sets the background color behind the text.

Example

.text{
background-color: #FFA500;
}
