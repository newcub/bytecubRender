What is CSS?

CSS stands for Cascading Style Sheet, and it is a styling language used for styling webpages.

CSS syntax:

Selectors:

A selector is the target element on a webpage that you want to apply the styling on.

Some examples of selectors are <p>, <h1>,<body>, <div>, <span>, <head> and so on...

Properties and Values:

Properties: Properties describe the aspects of the element you want to change, e.g color, font-size, height, width, etc.

Values: Values simply specify the setting you want to use for a chosen property. For example, if we wanted to change the color we might put red,blue,violet, etc.

Below is an example of how we can style the element, <p>This is a Paragraph</p> with CSS.

CSS

p{
color: blue;
}

Result

This is a Paragraph

Note:

Quantities in CSS are measured in px, %, rem, amd em. In this tutorial, we will mostly be using px and %.
