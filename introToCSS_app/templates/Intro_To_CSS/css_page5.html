Styling Lists

{%if has_answered == True%} {%else%} {%endif%}

Styling lists in CSS can enhance the visual presentation of your web content. Both odered lists [<ol>] and unordered lists [<ul>] can be customized using a variety of CSS properties.

5.1 Common CSS Properties for Lists

1. List Style Type

Define the bullet style for unordered lists or the numbering style for ordered lists.

Example

ul{
list-style-type: disc;
}
ol{
list-style-type: decimal-leading-zero;
}

2. List Style Position

Controls the possition of the bullet or number, either inside or outside the list item's content.

Example

ul{
list-style-position : inside;
}

3. Padding and Margin

Adjust the spacing around and within the list items.

Example

ul{
padding : 20px;
margin : 10px;
}

4. Custom Bullet Points

You can use images or custom shapes as bullet points.

Example

ul{
list-style-type : none;
}
ul li{
background-image : url('path/to/your/image.png');
background-size : 10px 10px;
background-repeat :no-repeat;
background-position : 0 5px;
padding-left : 15px;
}

Result

  • First
  • Second
  • Third

Note:

list-style-type: none; removes default bullets.

background-image holds the image.

background-size controls the size of the image.

background-position Adjusts the image's position.

padding-left Creates a space at the left side of the image.

5. Typography

Style the text within the list to be more visually appealing.

Example

li{
font-size : 16px;
color : #333;
line-height : 1.5;
}
