Table Styling

Styling tables in CSS can significantly enhance the visual appeal and usability of your data presentations.

Let's consider the HTML below:

HTML

Result

SN Name Age
01 Carl 60

Here are some key concepts an techniques to consider when styling HTML tables:

6.1 Common CSS Properties for Table Styling

1. Borders:

Use the border property to define the borders of the table and its cells.

Example

table{
border-collapse: collapse;
}
th, td{
border: 1px solid lightgray;
}

Result

SN Name Age
01 Carl 60
02 Kate 23

border-collapse: collapse; merges borders.

border: 1px solid lightgray; creates a lightgray border of 1px thickness around each cell.

2. Padding and Spacing:

Adjust the spacing within cells using padding and space between cells using border-spacing.

Example

table{
border-spacing: 5px;
}
th, td{
padding: 10px;
}

Result

SN Name Age
01 Carl 60
02 Kate 23

border-spacing: 5px; Creates a space of 5px between cells.

padding: 10px; Adds space of 10px inside cells.

3. Background Colors:

Use background-color to enhance readability.

Example

th{
background-color: lightgray;
}

Result

SN Name Age
01 Carl 60
02 Kate 23

4. Text Alignment:

Controls the alignment within cells using text-align.

Example

th{
text-align: left;
}
td{
text-align: center;
}

Result

SN Name Age
01 Carl 60
02 Kate 23

5. Hover Effects:

Improves interactivity by adding hover effects.

Example

tr:hover{
background-color: orange;
}

Result

SN Name Age
01 Carl 60
02 Kate 23
